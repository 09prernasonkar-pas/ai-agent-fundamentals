# AI Agent Fundamentals

## What is an AI Agent?
An AI Agent is a system that can understand a goal, plan steps, use tools, remember context, and produce outcomes autonomously.

Unlike a simple prompt, an AI agent can take actions and follow workflows.

---

## Agent vs Prompt

| Prompt | Agent |
|------|------|
| Single response | Multi-step execution |
| No memory | Has memory |
| No tools | Uses tools |

---

## Core Components of an AI Agent

1. LLM (Brain)
2. Instructions (System Prompt)
3. Tools (Search, APIs, Databases)
4. Memory (Short-term & Long-term)
5. Workflow (Control logic)

---

## Business Use Cases

### 1. Market Research Agent
An agent that gathers competitor data, analyzes trends, and produces summaries for product teams.

### 2. Policy Q&A Agent
An internal agent that answers employee questions using company policy documents.

---

## Why This Matters
AI agents are increasingly used in enterprise products to automate decision-making, improve efficiency, and reduce manual work.
