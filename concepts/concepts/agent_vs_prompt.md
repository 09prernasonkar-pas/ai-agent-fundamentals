# Agent vs Prompt

A prompt is a single instruction given to an LLM.
An agent is a system that can plan, act, and iterate.

## Key Differences

| Aspect | Prompt | Agent |
|------|------|------|
| Execution | One-time | Multi-step |
| Memory | No | Yes |
| Tools | No | Yes |
| Reliability | Low | Higher |

## Product Perspective
Agents are better suited for enterprise workflows where reliability and traceability are critical.
